Fix Freezes
    1. Check integrity of game files using steam
    2. If bug persits
        1. Reinstall csgo
        2. Remove "C:/Programm Files/Steam/userdata/<steamid>
        3. Install newest video drivers
        4. Update Windows if possible
        5. Reboot
